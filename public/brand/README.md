# Malaika Studios Logo Assets

Single-wing mark assets for Malaika Studios by Rotsi.

## Production Files

- `malaika-wing-mark.svg` - transparent charcoal/yellow wing mark for light backgrounds.
- `malaika-wing-mark-reverse.svg` - transparent white/yellow wing mark for dark backgrounds.
- `malaika-social-avatar.svg` - square charcoal social/profile avatar.
- `app/icon.svg` - Next.js favicon/app icon, using the same wing geometry.

## Usage Rule

Use the wing as a restrained signal of presence, care, and lift. Avoid adding halos, full angel wings, sparkles, religious framing, beauty-brand styling, or fake-premium gold effects.
